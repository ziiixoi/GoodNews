1. Install packages
-------------------------
npm i -D vite
-------------------------
npm i three
-------------------------

2. Run the development server
-------------------------
npm start
-------------------------

3. Build(create files for deployment)
-------------------------
npm run build
-------------------------



실행 정책을 일시적으로 Bypass로 변경하여 문제를 해결합니다:

Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
